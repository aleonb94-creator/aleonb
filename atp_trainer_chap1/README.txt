# ATP DGAC Trainer - Capítulo 01

Esta carpeta contiene una versión inicial del entrenador tipo Sheppard Air para el tema **Regulations**.

## Archivos
- `index.html` - aplicación principal
- `data/chapter01_regulations.json` - banco de preguntas extraído del PDF
- `manifest.webmanifest` - manifiesto para instalación como web app
- `sw.js` - cache offline básico
- `icons/` - iconos de la app
- `run_local_server.py` - servidor local simple

## Cómo usarlo
### Opción más fácil
1. Descomprime la carpeta.
2. Ejecuta `run_local_server.py`.
3. Abre en tu navegador: `http://localhost:8000`

### iPad o iPhone
1. Abre la URL local o súbelo a un hosting.
2. En Safari, usa Compartir -> Add to Home Screen.

### Laptop / escritorio
1. Abre la URL local o súbelo a un hosting.
2. En Chrome o Edge, instala la app desde el navegador si aparece la opción.

## Nota
La app guarda el progreso localmente en el navegador del dispositivo.
